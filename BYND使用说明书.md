# 第 0 章 · 开始之前：三个新朋友先认识一下

在动手之前，先花一分钟认识三个词。后面到处都会遇到它们。

### 🧠 API —— 手机的"大脑租赁服务"

BYND 这台手机本身**不会思考**。角色之所以能聊天，是因为手机把你的话送到远方的一台"AI 大脑"那里，大脑想好答案再传回来。
"API" 就是**连接那颗大脑的电话线**。你需要一条自己的电话线（第 3 章手把手教你接线）。

### 🤖 模型 —— 大脑的"型号"

同一家大脑公司会出很多种脑子：有的聪明但慢，有的快但笨一点。"模型"就是大脑的型号名，比如 `gemini-2.5-pro`、`deepseek-chat` 这样的字符串。接好电话线之后，你要从列表里挑一颗脑子。

### 🎴 角色卡 —— 装着一个"人"的图片

一张看起来普通的 PNG 图片里，可以藏着一个角色的全部人设：名字、性格、说话方式、开场白、背景故事。这种图叫**角色卡**（SillyTavern 格式，网上有大量分享）。把卡导入 BYND，角色就"住进"你的手机了。

> 💡 记不住也没关系，用大白话总结：**API 是电话线，模型是脑子，角色卡是人。**

# 第 1 章 · 把手机带回家（三种打开方式）

## 方式一：直接用浏览器打开（最简单，推荐所有人）

1. 打开手机或电脑上的浏览器（推荐 Chrome / Edge / Safari）。
2. 在地址栏输入：**`bynd.ccwu.cc`**，回车。
3. 看到一个锁屏界面？恭喜，你已经拿到手机了！

> ⚠️ 它是网页，**断网打不开**。想让页面更新到最新版，普通刷新一下就好（下拉刷新或 Ctrl+R），不需要任何复杂操作。

## 方式二：添加到主屏幕（像真 App 一样全屏，强烈推荐）

添加之后，桌面上会多一个叫 **BYND** 的图标，点开是全屏，没有浏览器地址栏，体验和真手机一模一样。

**安卓（Chrome / Edge）：**
1. 用浏览器打开 `bynd.ccwu.cc`；
2. 点浏览器右上角「⋮」菜单；
3. 找到「**安装应用**」或「**添加到主屏幕**」，点它；
4. 确认。桌面上出现 BYND 图标。

**iPhone / iPad（必须用 Safari）：**
1. 用 Safari 打开 `bynd.ccwu.cc`；
2. 点底部中间的「分享」按钮（一个带箭头的方框）；
3. 往下滑，点「**添加到主屏幕**」；
4. 点右上角「添加」。

> 🍎 iOS 用户特别注意：以后想用"角色主动发通知找你"的功能（第 18 章），**必须**先做这一步，直接在 Safari 里开网页是收不到通知的。

## 方式三：安卓 APK 安装包（进阶选项）

如果你拿到了 `BYND-xxx.apk` 安装文件（16MB 左右）：

1. 在手机上点开这个文件；
2. 系统会警告"来自未知来源"——点「仍然安装」/「允许」（这是因为它没有上架应用商店，不是病毒）；
3. 装好后桌面出现 BYND 图标。

APK 版的特点：
- ✅ 页面本身**离线也能打开**（但聊天、听歌等联网功能还是需要网络）；
- ✅ 文件导入、录屏等能力更强；
- ⚠️ **不会自动更新**——出了新版本需要重新下载安装包覆盖安装；
- 要求安卓 6.0 以上。

> 🤔 **选哪个？** 大多数人：方式一先玩起来 → 喜欢就用方式二装成"真 App"。方式三留给拿到安装包的用户。

# 第 2 章 · 第一次开机：锁屏和桌面

## 2.1 锁屏

打开 BYND，第一眼看到的就是锁屏，上面有：

- **大时钟**：真实的当前时间；
- **日期和天气**：天气是自动定位获取的（浏览器可能会问你"是否允许获取位置"，允许后才有天气）；
- **两张拍立得照片**：默认是占位图，以后可以换成你和 TA 的"合照"（第 15 章）；
- **底部滚动字幕**：可以放一句歌词（第 15 章）；
- **左右两个圆形按钮**：默认左边是手电筒（点了会发光，纯装饰），右边是相机。

**怎么解锁？→ 点一下屏幕上任意空白的地方就行**，不用滑动。

> 💡 嫌锁屏烦？可以关掉：美化 App → 全局壁纸设置 → 取消勾选「启用锁屏页面」，以后打开直接进桌面。

## 2.2 桌面

解锁后就是桌面。**左右滑动可以翻页**，默认有两页：

**第 1 页**：
- 顶部一个**日历**（纯展示，日期点不了）；
- 下面是 App 图标和两张照片小组件：**微信、正则、世界书、设置、美化、学习、记账、Game**。

**第 2 页**：**盗梦空间、监控、一起出门、PageMate、相册**。

**底部 Dock 栏**（翻页也不动的那一排）：**音乐、相机、预设**。

## 2.3 最顶上那一条（状态栏）

- 左边的时间是真的；
- 右边的**电量也是真的**（读的就是你手机/电脑的电池，充电时会变闪电图标）；
- WiFi 图标是装饰，不代表真实网络。

## 2.4 先记住两个 App 的位置

接下来两章会用到：

- **「设置」**（第 1 页，齿轮图标）——第 3 章去这里接电话线；
- **「微信」**（第 1 页，绿色图标）——第 4 章去这里领角色。

# 第 3 章 · 最重要的一步：给手机装上"大脑"（API 设置）

> 🌟 **这是整本说明书最重要的一章。** 没有这一步，角色一句话都说不出来。
> 好消息是：只需要做一次，做完就一劳永逸。

## 3.1 你需要先有一把"钥匙"

要接上 AI 大脑，你需要从"卖大脑服务的站点"拿到三样东西：

| 你需要的 | 长什么样 | 大白话 |
|---|---|---|
| **接口地址（Base URL）** | `https://某某网站.com/v1` | 大脑公司的"门牌地址" |
| **API Key（密钥）** | `sk-` 开头的一长串字母数字 | 进门的钥匙，**别给别人看** |
| **模型名** | 不用记，到时候从列表里选 | 脑子型号 |

**去哪里拿？** 常见渠道：
- 各种 **OpenAI 兼容中转站**（搜索"AI API 中转"能找到很多，有免费有付费）；
- DeepSeek、硅基流动等国内官方平台（注册就送额度）；
- 你购买产品时商家推荐的渠道。

> ⚠️ 只支持 **"OpenAI 兼容格式"** 的接口（绝大多数中转站都是）。Claude 官网、Gemini 官网的"原生接口"不能直接填——但没关系，中转站转过的 Claude 和 Gemini 都能用。

## 3.2 手把手接线（跟着点）

1. 桌面点 **「设置」**；
2. 最顶上就是 **「API 连接」** 区，点 **「添加 API」** 按钮；
3. 弹出表单，逐个填：
   - **名称**：随便起，比如"我的大脑1号"（框里的提示语是"给它取个名字吧~"）；
   - **Base URL**：粘贴你拿到的门牌地址；
   - **API Key**：粘贴你的钥匙（有些免费站不需要钥匙，可以空着）；
   - **模型**：现在还是灰色点不了？对的，**先别管它**；
4. 点 **「测试」** 按钮，等几秒钟：
   - 看到绿色的 **「✅ 连接成功！(N 个模型，可聊天: xxx)」** → 太棒了，下一步；
   - 看到红色的 ❌ → 翻到本章 3.5 的"报错翻译"对照着修；
5. 现在 **「模型」下拉框能点了**——点开，选一个模型（测试时系统已经自动把"确认能聊天"的模型排在了最前面，选第一个准没错）；
6. 点 **「保存」**。

🎉 完成！你添加的第一个 API 会自动成为"默认大脑"。

## 3.3 认识 API 卡片和"皇冠"

保存后，API 连接区会出现一张卡片：

- 左边的**小圆灯**：绿色 = 在线，红色 = 上次测试失败，黄色 = 测试中；
- 名字旁边的 **♛ 皇冠** = 这是当前聊天用的默认大脑；
- 卡片右边四个小按钮：
  - 📡 **重新测试**；
  - ⭐ **设为聊天默认**（有多个 API 时用来切换）；
  - 🖼 **设为生图默认**（想让角色"发照片"给你，需要一个有生图模型的 API）；
  - 🗑 **删除**（会弹确认框）。

卡片列表上方还有个 **「API 工作台」**面板——聊天、生图、语音三条线各用哪个 API，在这里一目了然、直接切换。

## 3.4 想让 TA 有声音？（语音 API，可选）

在同一个设置区找到**语音 API** 卡片，支持四种服务商：MiniMax / OpenAI / ElevenLabs / 本地 TTS。以 MiniMax 为例：点「配置」→ 填 API Key 和音色 ID（弹窗里自带接入教程）→ 点「测试语音」听一下 → 「设默认」。
这是可选项，不配语音也完全不影响聊天。

## 3.5 报错翻译（红色 ❌ 对照表）

| 屏幕上显示 | 人话 | 怎么办 |
|---|---|---|
| `API Key 无效 (401)` | 钥匙不对 | 检查 Key 有没有复制全、过没过期 |
| `没有权限 (403)` | 钥匙对了但这扇门不让进 | 问服务商你的 Key 有没有开通这个模型 |
| `HTTP 404` | 门牌地址写错了 | 检查 Base URL（结尾一般是 `/v1`） |
| `连接超时` | 敲了 10 秒门没人应 | 检查网络；换个时间再试；或换一家站点 |
| `这个接口可能不允许浏览器直连…` | 这家店不接待"网页"客人（跨域限制） | 换一家支持浏览器直连（CORS）的中转站 |
| `请先填写 Base URL` | 地址栏空着呢 | 填上再点测试 |

> 💡 **"获取模型列表失败，可手动输入"**——有些站点不给列表但其实能用，这时模型框会变成手动输入框，把服务商告诉你的模型名照着敲进去就行。

# 第 4 章 · 领养你的第一个角色

## 4.1 方式一：导入角色卡（推荐）

你手里有一张 PNG 角色卡？三十秒让 TA 住进来：

1. 桌面点 **「微信」**；
2. 点右上角 **「+」** → 选 **「导入角色」**；
3. 底部弹出菜单，选 **「导入角色卡 (PNG)」**；
4. 选中你的 PNG 文件 → 页面自动解析，弹出"解析成功"卡片，名字和开场白都自动填好了；
5. 接着会进入**头像裁剪**：拖动、缩放选好头像范围；
6. 点 **「确认添加」**。

TA 出现在聊天列表里了！角色卡里自带的**备选开场白、世界书、正则脚本**也会一起被搬进来（分别去对应 App 里能看到）。

> ❌ 提示"无法识别，请确保是 PNG 格式的角色卡"？说明这张图只是普通图片，里面没藏人设数据。找卡的时候认准"SillyTavern 角色卡 / PNG 卡"字样。

## 4.2 方式二：手动捏一个

1. 微信右上角 **「+」** → 「导入角色」→ 选 **「手动新建角色」**；
2. 上传头像（可裁剪）、起名字（必填）、写一句开场白；
3. 保存。

手动建的角色人设比较素，想让 TA 更立体，可以之后在**聊天设置 → 角色简介**里补充人设，或去**世界书**给 TA 写设定（第 12 章）。

## 4.3 换开场白 / 删除角色

- **备选开场白**：有些卡自带好几个开场剧本，在聊天里可以左右切换选一个你喜欢的开局（切换不会弄丢聊天记录）；
- **删除角色**：聊天列表里把那一条**向左滑**，出现红色「删除」→ 确认。⚠️ 聊天记录会一起消失，无法恢复。

# 第 5 章 · 开始聊天！（基础操作）

点开聊天列表里的角色，进入聊天室。底部输入栏从左到右是：
**🎤 语音 | 输入框 | 🖼 图片 | 📷 相机 | 😊 表情 | ➕ 更多 | 发送 | ✨AI回复**

## 5.1 发消息

- 打字 → 点纸飞机（或回车）发送；
- TA 回复时会显示"对方正在输入…"，请耐心等几秒到几十秒（取决于你的大脑快不快）；
- TA 的一次回复往往是**好几条气泡**连发——这是特意设计的，像真人发微信一样。

## 5.2 ✨ AI 回复按钮（很常用！）

最右边的星星按钮：**不发消息，直接让 TA 接着说**。
用途：TA 说完一段你想让 TA 继续说；或你刚改了设定想让 TA 立刻表现一下。

## 5.3 发图片和拍照

- **🖼 图片**：从相册选，**可以多选**——连选几张会自动叠成一摞"图片堆"，聊天里可以左右翻着看；
- **📷 相机**：现场拍一张直接发；
- 如果你的模型支持"看图"（视觉模型），TA 能看懂你发的照片并评论它。

## 5.4 表情和贴纸

点 😊 打开表情面板：
- 顶部有多个**贴纸包标签**：内置包、你自建的包，还有 **GIPHY 在线搜 GIF**（直接输中文就能搜）；
- 点任意贴纸 = 直接发送；
- 微信主题下还有经典**小黄脸**（[微笑][捂脸]那种，插进文字里一起发）；
- 面板右上角齿轮 = 管理贴纸包（建包、导入，详见 9.6）。

## 5.5 语音消息

点 🎤，面板顶部可以切换两种模式：
- **按住说话**：真的录音！按住说、松手发（最长 60 秒），会自动转文字，还会分析你的语气告诉 TA（你是开心地说的还是委屈地说的，TA 听得出来）；
- **输入文字**：打字 → 点「发送语音」，气泡显示成语音条的样子。

收到 TA 的语音：点气泡播放（配了语音 API 才有真声音），可展开看"转文字"。

## 5.6 长按消息（超实用菜单）

**长按任何一条消息**（约半秒），弹出菜单：

| 菜单 | 干嘛的 |
|---|---|
| **编辑** | 改这条消息的文字（改 TA 的话也可以！） |
| **引用** | 回复时带上这条消息 |
| **删除** | 删掉这一条 |
| **收藏** | 存进「我 → 收藏」（仅 TA 的消息） |
| **重新roll** | ⭐ 不满意 TA 这轮回复？点它重新生成一遍 |
| **多选删除** | 批量勾选删除 |

## 5.7 点 TA 的头像（两个彩蛋）

- **单击头像** = 打开「**状态小票**」：一张小收据，写着 TA 此刻的内心独白、想法、穿着、动作、目光……每次 TA 回复后自动更新，右上角还能翻历史记录；
- **双击头像** = 打开「**TA 的手机**」——没错，你可以查 TA 的手机（第 9.7 节详述）。

# 第 6 章 · 聊天进阶：红包、打电话、一起听歌……

点输入栏的 **➕**，打开工具面板，这里藏着一堆好玩的：

## 6.1 转账和红包 💰

- **转账**：填金额（必填）和说明 → 发送。TA 会对你打钱这件事做出反应（惊讶、推辞、感动……看人设）；
- **红包**：标题默认"恭喜发财，大吉大利"，金额可以空着 → 发送；
- **TA 也会给你发**：点 TA 发来的红包/转账 → 可以选「收下」或「退回」——你的选择 TA 都知道，会有后续反应；
- TA 还可能送你**礼物**，甚至给你开**亲密付**（TA 帮你付钱的额度，购物时能用，见 9.4）。

## 6.2 拍一拍 和 震动

- **拍一拍**：像真微信那样"你拍了拍对方"，文案可以自己改；
- **震动**：发出后你的屏幕真的会晃一下，文案也能改。

## 6.3 语音通话 / 视频通话 📞

1. ➕ 面板选「语音通话」或「视频通话」（联系人资料页也有入口）；

4. **视频通话**的特别之处：接通后会用生图 API 生成"TA 的摄像头画面"——你真的能看到 TA！在聊天设置里还能上传通话背景、打开你自己的真实前置摄像头（画面显示在右下角小窗）；

## 6.5 共享屏幕给 TA 看 🖥

➕ 面板选「共享屏幕」→ 浏览器询问共享哪个窗口 → 确认后 TA 能"看到"你的屏幕并聊它（比如陪你一起看视频、看你打游戏）。

# 第 12 章 · 世界书：给 TA 的世界写设定

**世界书是什么？** TA 的"设定集"。你们故事里的世界观、重要人物、你的习惯、你们的纪念日……写进世界书，TA 每次说话前都会"复习"一遍这些设定。

入口：桌面 → 「**世界书**」（书本图标）。

## 12.1 书架

打开是一个超美的乐谱书架：**每个角色一本书**，封面就是 TA 的头像，封面角上的数字是设定条数。左右滑动书架时书会沿曲线滚动，还会叮叮咚咚**弹钢琴**（对，这是真的）。

点一本书 → 开书动画 → 进入这本书。

## 12.2 加设定

1. 书内页点右上角 **「+」**；
2. 填 **关键词**（相当于这条设定的标签，如"帝国, 魔法"）和 **内容**（正文，必填）；
3. 「保存条目」。

每条设定右侧有个**开关**：开 = TA 能"看到"这条设定；关 = 暂时屏蔽。

> 📌 小知识：这里的"关键词"只是标签，**不管聊没聊到关键词，开着的条目 TA 都能看到**。所以设定不用堆太多，捡重要的写。

## 12.3 两个进阶玩法

- **在书架页点「+」= 新建一本书**，同时会**在微信里创建一个同名新角色**（从设定开始捏人的玩法）；
- **贴纸彩蛋**：在世界书条目内容里按"名字：图片链接"的格式写图片，这些图会变成 **TA 专属的贴纸包**，TA 聊天时会拿来发！

## 12.4 和聊天的配合

每个聊天的设置面板里有「聊天世界书」勾选列表——可以精细控制哪些条目在这个聊天里生效（默认全生效，不用管）。

# 第 13 章 · 正则脚本：消息的"显示滤镜"

**正则是什么？** 一个"显示滤镜"App：AI 的回复在显示到屏幕上之前，先经过你定的规则加工——可以**隐藏**某些内容（比如 AI 的思考过程）、**替换**固定格式、或把代码**渲染**成漂亮样式。它只改"你看到的"，不改存档，也不改发给 AI 的内容。

入口：桌面 → 「**正则**」（代码图标）。顶部两个标签：

- **全局**：规则应用到所有聊天。出厂自带一条「Markdown 加粗」（把 `**文字**` 显示成粗体）；
- **角色**：只作用于某一个角色（顶部选角色）。角色卡自带的正则脚本会出现在这里。

**新建规则**：右上角「+」→ 选作用域 → 起名 → 填「正则」（匹配什么）和「替换」（换成什么，留空=直接删掉匹配内容）→ 保存。每条规则有开关，即时生效，全局和角色规则都会永久保存。

> 🔰 不会写正则？没关系——这个 App 主要服务两种人：用了自带正则的角色卡的人（自动生效，你什么都不用做），和爱折腾的高级玩家。普通玩家可以整个跳过本章。

# 第 14 章 · 预设：决定 TA 怎么说话

**预设是什么？** 一套"说话风格指导书"——文风是甜是虐、回复长短、剧情节奏，都由预设控制。玩角色扮演的圈子里流传着大量现成预设文件（.json）。

Dock 栏 → 「音乐」。可以搜歌、播放、看歌词、收藏，还能和角色一起听（第 6.4 节）。

## 16.1 网易云歌单搬家

主页搜索框下方有「**网易云音乐**」入口卡：

1. 点进去自动出现**登录二维码**；
2. 打开手机上的网易云音乐 App → 扫一扫 → 手机上点「确认登录」（二维码过期就点"刷新"）；
3. 登录后能看到你的"我喜欢的音乐"和全部歌单；
4. 点开歌单 → 「**导入到音乐**」，以后在音乐 App 主页直接听，不用再进网易云界面。

**什么都不用安装、不用配置**，开箱即用。（设置里那两个"API 服务"输入框请**保持空着**，那是给自建服务的高级玩家的。）

## 16.2 几个提醒

- 会员歌曲能不能听，取决于**你扫码的那个网易云账号**自己有没有会员；
- 提示"这首歌没拿到播放地址（可能是数字专辑）"→ 版权限制，换首歌吧；
- 登录凭证只存在你自己的设备里，不会上传给任何人。

# 第 18 章 · 让 TA 主动来找你（消息提醒）

想体验"手机一震，是 TA 发消息来了"？

1. 设置 App → 「**主动消息提醒**」卡片；
2. 打开「后台消息」开关 → 浏览器询问通知权限 → **允许**；
3. 选提醒时间：30 分钟 ~ 24 小时，或自定义；
4. 点「测试通知」验证一下能不能收到。

之后只要你太久没理 TA，就会收到系统通知：「**XX 想找你**——已经等你 24 小时了」。点通知直接跳回聊天。

**注意事项：**
- 🍎 **iOS 必须先"添加到主屏幕"**（第 1 章方式二）再开通知，否则收不到；
- 网页版的提醒在**浏览器还活着**（开着或后台）时有效；想关掉浏览器甚至关机后仍能收到，需要配置云端推送（设置里有"主动消息新手教程"弹窗，跟着走）；
- 🥚 彩蛋：TA 如果嫌你设的间隔太久，**会自己动手改短**……设置页里会显示"上次由 XX 改成这个时间，理由是：……"。

**桌面消息岛**：不在聊天页时 TA 来消息，屏幕顶部会弹出灵动岛通知（带提示音），点击直达——这个不用设置，天生就有。

# 第 19 章 · ⚠️ 保命章：备份你的数据（必读！）

> **请把这一章读两遍。**

BYND 没有账号系统、没有云存档——你的一切（角色、聊天记录、记忆、设置、美化）都存在**这台设备的这个浏览器**里。这意味着：

| 做了这件事 | 后果 |
|---|---|
| 清除浏览器缓存/浏览数据 | 💥 **全部消失** |
| 卸载浏览器 / 恢复出厂 | 💥 **全部消失** |
| 换新手机 | 数据不会跟过去 |
| 用无痕模式打开 | 关掉就没 |

## 唯一的保险：定期导出

1. 设置 App → 「数据管理」→ 「**导出数据**」；
2. 会下载一个 `AI_OS备份_日期.json` 文件——这就是你们的全部回忆，**收好它**（存网盘、发自己邮箱都行）；
3. 恢复 / 搬家到新设备：新设备打开 BYND → 设置 → 「导入数据」→ 选备份文件 → 确认覆盖 → 自动刷新，一切如初。

> 💡 建议：**每周导出一次**；聊出重要剧情后随手导一次。备份文件里含你的 API 钥匙，别发给陌生人。
>
> 🧹 「清理无效缓存」按钮是安全的——它只清垃圾，不动你的角色和聊天。

# 第 20 章 · 出问题了怎么办

## 20.1 TA 不回复 / 报错了

看聊天里的红色错误提示，对照翻译：

| 提示 | 人话 | 怎么办 |
|---|---|---|
| `还没有设置 API 哦～` | 电话线没接 | 回第 3 章 |
| `还没选模型哦～` | 线接了但没选脑子 | 设置 → 编辑 API → 测试 → 选模型 |
| `API 错误 (401)` | 钥匙不对 | 检查 API Key |
| `API 错误 (404)` | 地址或模型名不对 | 检查 Base URL 和模型 |
| `API 错误 (429)，已暂停自动请求约 X 分钟` | 聊太快，被大脑公司限流 | 等几分钟自然恢复，不用动 |
| `API 错误 (500/502/503)` | 大脑公司那边故障 | 稍后再试或换一个 API |
| `请求超时（90秒）…` | 大脑太慢 | 换更快的模型，或精简角色卡/世界书 |
| `AI 返回了空内容，可能被模型安全策略拦截` | 内容被大脑公司过滤了 | 换个说法，或换个宽松点的模型 |
| 提到 `image/vision` | 当前模型看不懂图 | 换支持"视觉"的模型再发图 |
| `AI 这次只返回了思维链，已拦截` | TA 光想没说，系统自动重试中 | 等一下就好 |

## 20.2 常见问题 FAQ

**Q：页面好像不是最新版本？**
A：普通刷新一下（下拉刷新 / Ctrl+R）就好。顽固时 Ctrl+Shift+R 强刷。BYND 不搞缓存陷阱，刷新即最新。APK 版需要重装新安装包。

**Q：TA 一次回复要等多久？**
A：几秒到几十秒都正常，取决于你的 API 快慢。有时系统会自动重试（回复被截断时自动续写），会显得久一点。

**Q：我发的消息 TA 记得多久？**
A：TA 直接"看得见"最近约 30 条消息；更早的靠记忆星河（第 8 章）保存精华。重要的事写进记忆或世界书最保险。

**Q：语音、生图都要单独的 API 吗？**
A：聊天 API 必须有；生图可以用同一个 API 里的生图模型或单加一个；语音要单独配语音服务。只配聊天也完全能玩。

**Q：数据会上传到哪里吗？**
A：角色和聊天记录全在你设备本地。聊天时对话内容会发给**你自己填的那个 API** 处理（这是 AI 能回复你的原理）。

**Q：换浏览器 = 换手机吗？**
A：是的！同一台电脑上 Chrome 和 Edge 是两台"手机"，数据不互通。搬家用导出/导入（第 19 章）。

**Q：手机浏览器里怎么全屏？**
A：第一次触摸屏幕会自动请求全屏；想要最完美的全屏体验请"添加到主屏幕"（第 1 章方式二）。
